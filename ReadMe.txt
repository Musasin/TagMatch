◇ 起動方法 ◇
  Mac OS : 同フォルダ内のゆかマキタッグマッチ!!.appから起動してください。
  Windows: 同フォルダ内のゆかマキタッグマッチ!!.exeから起動してください。

◇ 削除方法 ◇
	インストールを行う方式では無いため、そのままフォルダごと削除して下さい。

◇ 操作方法 ◇
	キーボードで操作します。 ゲーム内でキー設定の変更が可能です。
	左右キー: 移動
	下キー: しゃがみ
	ESCキー: メニュー開閉
	Cキー: ジャンプ, 決定
	Xキー: ショット, キャンセル
	Zキー: スイッチ(キャラクター切り替え)
